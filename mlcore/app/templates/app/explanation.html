Arquitectura del Sistema Inteligente

1. Flujo de Datos (Pipeline)

El sistema sigue un proceso lineal y robusto:

  1. Ingesta: Recepción de archivos CSV/Excel mediante Django Forms.
  2. Preprocesamiento: Limpieza de valores nulos e imputación estadística con Pandas.
  3. Codificación: Transformación de variables categóricas (texto) a numéricas usando LabelEncoder.
  4. Normalización: Escalado de datos con StandardScaler para mejorar la convergencia del entrenamiento.

2. Red Neuronal Backpropagation

Se utiliza una red Multicapa (MLP) construida con TensorFlow:

  • Capas Ocultas: 2 capas de 32 neuronas cada una con función de activación ReLU para manejar la no-linealidad.
  • Capa de Salida: 1 neurona con función Sigmoid para clasificación binaria (0 o 1).
  • Algoritmo: Se utiliza el optimizador Adam, que es una evolución del Descenso de Gradiente Estocástico, para realizar el Backpropagation automático ajustando los pesos y sesgos.

3. Evaluación de Resultados

Para validar el modelo, el dataset se divide en 80% entrenamiento y 20% prueba:

  • Accuracy: Mide el porcentaje total de predicciones correctas.
  • Matriz de Confusión: Permite visualizar los Falsos Positivos y Falsos Negativos, crucial para entender dónde se equivoca la red.
  • Loss: Indica qué tan lejos está la predicción del valor real durante el entrenamiento.

Nota para el examen: Si el profesor pregunta por el "Backpropagation", recuerda mencionar que es el proceso donde se calcula el error en la salida y se propaga hacia atrás para ajustar los pesos usando la derivada de la función de activación.
