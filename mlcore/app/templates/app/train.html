Instrucciones Académicas
  1. Sube un archivo en formato CSV o Excel (.xlsx).
  2. El sistema detectará automáticamente el target (columna 'survived' o la primera).
  3. Se realizará limpieza de datos (imputación de nulos).
  4. Se codificarán variables categóricas (LabelEncoding).
  5. Se entrenará un modelo con 2 capas ocultas de 32 neuronas cada una.
  6. Los resultados se mostrarán en gráficos interactivos.
Tip: Usa entre 50 y 200 iteraciones para ver mejores resultados en datasets pequeños como el Titanic.
